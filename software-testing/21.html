Automated Software Test

Here, we test 021 - Do the relevant assets exist within the game files?
This ensures that all assets required for the game are present.

This was tested via automated testing using LibGDX Testing, created by TomGrill.

We used the following methods in the test, which are then checked by a parent method to determine a pass or fail:
- mapAssetsExist() - Checks all map assets exist in the game.
- skinAssetsExist() - Checks all character assets exist in the game.
- audioAssetsExist() - Checks all audio files are present.
- imgAssetsExist() - Checks all other in-game images are present.
- imgMenuAssetsExist() - Checks all assets relating to in-game menus exist.
